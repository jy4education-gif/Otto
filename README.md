# Gedicht — Ottos Mops

This is a minimal HTML page that displays the poem from `gedicht.txt`.

How to view

1. Open `index.html` in your browser (double-click or drag into browser).
2. Use the "Download" button to save the poem as a text file.
3. Use the "Copy" button to copy the poem to clipboard.

Files added:

- `index.html` — main page
- `styles.css` — styling
- `script.js` — small helpers (download/copy)
